What is happening?

The website is made of two parts, the frontend and the backend. When you login on the frontend it sends your username and password to the backend server. This then logins into gradespeed using those credentials, and grabs your semester average for each class. This is then sent back to the client, which calculates the GPA. During this process all data is stored in memory and never written to disk or any long term storage medium. All data is deleted after the server responds to the client.

Logging?

This service is completly log free with no plans of ever implementing logging as we believe in complete privacy. All data is deleted the minute the server is done using it
